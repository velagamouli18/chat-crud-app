# Chat CRUD App

A full-stack web application built using Node.js, Express, MongoDB, and EJS.

## Features
- Create, view, edit, and delete chat messages
- RESTful routing using Express
- MongoDB integration with Mongoose
- Server-side rendering using EJS

## Tech Stack
- Node.js
- Express.js
- MongoDB
- EJS

## How to Run
1. Clone the repo
2. Run `npm install`
3. Start MongoDB locally
4. Run `node index.js`
5. Open http://localhost:3000/chats
